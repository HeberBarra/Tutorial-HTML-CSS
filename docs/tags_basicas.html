Estrutura Inicial de Todo Arquivo HTML

É de bom costume que todo arquivo HTML comece uma tag <!DOCTYPE html> para indicar que a versão do HTML utilizado é a 5. Em seguida deve ser criada a tag <html>, que deve ser fechada ao final do arquivo, na abertura dessa tag deve indicar a língua do site — para questões de acessibilidade e afins — para a nossa língua materna basta colocar lang="pt-br" dentro da tag <html>, dessa forma: 

<html lang="pt-br">

lang é um atributo, conforme formos vendo e utilizando as tags, conheceremos mais atributos. Dentro da tag <html> criaremos a tag <head> que serve para a parte técnica do site, como título, ícone da aba, algumas configurações de display, e outras coisas que veremos mais adiante. Depois da tag <head> criaremos a tag <body> que é onde fica o conteúdo do site, mas ainda podemos colocar css e scripts dentro do body sem problema algum. Temos que definir o charset(conjunto de caracteres) que será utilizado pelo site, normalmente utilizamos o UTF-8, fazemos da seguinte forma: 

<meta charset="UTF-8">
Também devemos definir o título da paǵina: 
<title>Título da Página</title>
A seguinte tag ajudará com a questões de exibição 
<meta name="viewport" content="width=device-width, initial-scale=1.0" >

Estas três tags devem ficar dentro da tag <head>, depois de fazer tudo isso o resultado é o seguinte: 


<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título da Página</title>
    </head>
    <body>                
    </body>
</html>

Conteúdo da Página

Agora veremos como inserir conteúdo no site.

Tags Textuais

Para inserir textos que apareceram no site, basta digitá-los dentro da tag <body>, mas fazer isso sem colocar uma tag antes pode causar problemas em relação a estrutura do site. A tag <p> serve para criar parágrafos, para utilizá-la é simples, basta fazer da seguinte maneira: 

<p>Texto</p>

Outra tag que pode utilizada para inserir textos é a tag <span>, que criar um contêiner genérico, com display em linha(veremos depois), que pode ser usado para outras coisas além de texto, semelhantemente a tag <p>, para inserir textos usando a tag <span> fazemos da seguinte forma: 

<span>Texto</span>

As tags textuais demonstradas até agora serve para a criação de blocos de textos, mas o HTML possui tags específicas para a criação de títulos, com uma hierarquia entre eles, estas são as tags de cabeçalho, que vão de <h1> até <h6>, com <h1> sendo a mais importante. Utilizamos ela da seguinte forma: 


<h1>Título Principal</h1>
<h2>Título Secundário</h2>
<h6>Título com Menos Importância</h6>
Resultado:

Existem também tags que são normalmente utilizadas para estilizar o texto(também representam diferenças semânticas para aquele trecho), para deixar itálico, negrito, etc.. Podemos recriar o efeitos dessas tags usando CSS, mas nada impede de usá-las para pequenos trechos de textos, como uma palavra ou frase. Exemplos: 


<i>itálico</i>
<b>negrito</b>
<del>texto excluído</del>
Resultado: 
itálico
negrito
texto excluído

Tags de Mídia

Além de textos, podemos inserir várias formas de mídias no site, como fotos, vídeos, áudios e até mesmo outras páginas. Primeiramente veremos como inserir imagens, para isto usaremos normalmente a tag <img>,diferentemente da maioria das outras tags, esta tag não precisa ser fechada. Para indicar qual imagem que deve ser exibida devemos usar o atributo src, que recebe o caminho ou a url até a imagem. No caso do caminho até a imagem o ideal é sempre colocar a imagem junto dos arquivos do site e usar o caminho relativo até a imagem. Por exemplo, considerando a seguinte estrutura de arquivos: 


Exemplo/
├── css
├── imagens
│   └── exemplo.png
└── index.html

Agora, para colocar a imagem exemplo.png no index.html basta colocar dentro dele a tag <img> da seguinte maneira: 

<img src=”imagens/exemplo.png” alt=”imagem de exemplo”>
Resultado:

Além de imagens, podemos inserir áudios usando a tag <audio>, assim como a tag <img> precisamos fornercer um valor para src, sendo a url ou caminho até o áudio. Precisamos controlar o atributo controls para que o navegador mostre os controles. Considere a seguinte estruta: 


Exemplo/
├── audio_exemplo.mp3
└── index.html
No index.html, colocamos a tag <audio>: 
<audio src="audio_exemplo.mp3"> controls</audio>
Resultado:

Para a inserção de vídeos usamos a tag <video>. Estrutura de arquivos: 


Exemplo/
├── video_exemplo.mp4
└── index.html
No arquivo index.html: 
<video src="video_exemplo.mp4"></video>
Resultado:

A tag <iframe> permite a exibição de outra página, até mesmo de outro site(alguns sites podem não permitir a exibição deles usando um iframe), na página atual, ocupando apenas uma porção da página(ou a página inteira, se assim for desejado). Exemplo: 

<iframe src="iframes/headings.html">

Outras tags

Por último veremos algumas tags, que são extremamente úteis. Primeiro a tag <div> que é similar a tag <span>, mas possui um display em bloco, e também é utilizada comumente para criar divisões na página. Exemplo: 


<div>
<h4>Título de Exemplo</h4>
<p>Texto de Exemplo</p>
</div>
Outra tag muito importante é a tag <a> que é usado para criar hiperlinks, assim como as tags de mídia essa tag precisa de um atributo para funcionar corretamente, que nesse caso é o atributo href, que recebe o caminho até um arquivo ou a url que será usada no hiperlink. Também a tag <a> precisa de algum conteúdo dentro dela que será transformado no hiperlink. Exemplo: 

<a href="https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/a">
    a tag <a> possui algumas outras utilidades que podem ser vistas neste link
</a>
Resultado: a tag <a> possui algumas outras utilidades que podem ser vistas neste link

A última tag "básica" é a tag <button> que permite a criação de botões clicáveis, que com apenas HTML e CSS são meio inúteis sendo utilizados apenas em formulários, mas com Javascript e outras linguagens de programação, os botões podem ter mais utilidades. Exemplo: 


<button>Botão Sem Utilidade</button>
